## 📋 Sobre o projeto

Landing page profissional desenvolvida do zero para uma construtora especializada no programa **Minha Casa Minha Vida**. O projeto passou por todas as etapas de desenvolvimento:
Análise de negócios → Requisitos → Design → Código → Deploy

text

## 🎨 Design

| Elemento | Detalhe |
|----------|---------|
| Layout | Bento Grid (estilo Apple) |
| Tipografia | Archivo Black + Hind |
| Paleta | Amarelo `#F5C518` · Preto `#0A0A0A` · Gelo `#F5F3EE` |
| Estilo | Moderno, minimalista, alto contraste |

## ✨ Funcionalidades

- **Navegação fixa** com fundo escuro
- **Hero section** com tipografia gigante (96px)
- **Bento Grid** com imagens e pilares intercalados
- **Seção Sobre** com fundo preto
- **Lista de serviços** numerada e organizada
- **Números de impacto** (+100 empreendimentos)
- **Botão WhatsApp** com animação pulse
- **Totalmente responsivo** (mobile + desktop)
